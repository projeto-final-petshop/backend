# Pet Conncet

## Pré requisitos

* JDK 17
* Maven
* MySQL
* Client HTTP (Postman, Insomnia, cURL etc.)

## Iniciando a aplicação

Comandos para Executar a aplicação no Terminal

* Rodar a aplicação: `mvn spring-boot:run`
* Instalar dependências: `mvn install`
